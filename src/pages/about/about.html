What is a Wi-Lock and why?

Wi-Lock is a mobile application that being used to unlock or lock the door. User need to be within specific range in order to request Real-time Authentication Code (RAC) number. All the procedure is listed down below:

1. Make sure to be within the specific range.

2. Request RAC number to enabling the verification button.

3. Verify the RAC number.

4. After a the number successfully being verified, you are now have access to lock or unlock the door.

Enjoy using this Wi-Lock System :)
